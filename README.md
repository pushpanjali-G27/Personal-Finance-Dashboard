# Personal-Finance-Dashboard
The Personal Finance Dashboard is an interactive Excel project designed to analyze income, expenses, and savings. Using Pivot Tables, charts, and slicers, it provides clear financial insights and spending trends. This project demonstrates data cleaning, data analysis, and visualization skills to support better budgeting and financial decisions.
